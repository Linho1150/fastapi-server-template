# fastapi-server-template
## Setting virtual environment

- create enviroment
  ```bash
  python3 -m venv projectName
  ```

- access enviroment
  ```bash
  source projectName/bin/activate
  ```

- install package
  ```bash
  pip install -r requirements.txt
  ```
  
- export package list
  ```bash
  pip freeze > requirements.txt
  ```
